# Lisse


Very simple command line Operating System.
<br>
Feel free to commit some code.
This os does not support UEFI mode only legacy.
### How to run:
 - qemu-system-i386 -cdrom Lisse.iso
 - OR
 - burn to disc
 
### Commands
 - clear
 
### How to compile
 - run build.sh

### What you will need
 - gcc
 - ld
 - grub-mkrescue
 - nasm
 
### Features
 - one command
 - x86 
 - keyboard
 - interrupts
