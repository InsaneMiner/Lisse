### how to run:
 - qemu-system-i386 -cdrom Lisse.iso
 - OR
 - burn to disc

This os does not support UEFI yet.
<br>
### commands
 - clear
### How to compile
 - run build.sh
<br>

### What you will need
 - gcc
 - ld
 - grub-mkrescue
 - nasm
### Features
 - one command
 - x86 
 - keyboard
 - interrupts
